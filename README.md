# Arquetipos de Marca

Herramienta docente que guía paso a paso la construcción del arquetipo de marca
de un proyecto, basada en Mark & Pearson (2001), *The Hero and the Outlaw*.

**En producción:** https://arquetipos.kimeltun.app

## Contenido

| Archivo | Qué es |
|---|---|
| `index.html` | Portada: explica el marco, los doce arquetipos y las limitaciones |
| `arquetipos.html` | La app: 14 decisiones y ficha descargable en PDF |
| `img/` | Las doce ilustraciones (generadas con IA para este proyecto) |
| `CNAME` | Dominio personalizado para GitHub Pages |
| `LICENSE` | CC BY-NC-SA 4.0 |

Sitio 100% estático. No requiere servidor, base de datos ni build. Todo corre en
el navegador y no se transmite ningún dato.
